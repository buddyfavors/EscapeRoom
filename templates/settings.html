Settings

Edit lock code pools as JSON. Keys: digit3 (3 digits), letter5 (5 letters), digit4 (4 digits). Hard mode needs at least four 3-digit, two 5-letter, and four 4-digit entries.

RFID tags

One 10-digit tag ID per line (leading zeros count). Blank lines and lines starting with # are ignored. When a listed tag is scanned the engine randomly decides whether it's a good scan (reveals one character on a random unsolved lock) or a bad scan (Gamemaster punishment). Harder difficulties bias the roll toward bad. Each tag can be used once per run.

Wheel of punishments

After three wrong lock combinations in a row, the wheel spins and one line below is chosen at random. Lines starting with minigame launch a minigame (use minigame for random, or minigame:<slug> for a specific one — reaction-rush, whack-mole, rps, simon, pattern). Any other line is shown to the players as a text punishment.

GM — live session

Shows active combinations for the running game (keep off the player monitor).
