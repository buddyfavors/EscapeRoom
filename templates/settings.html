Settings

Edit lock code pools as JSON. Keys: digit3 (3 digits), letter5 (5 letters), digit4 (4 digits). Hard mode needs at least four 3-digit, two 5-letter, and four 4-digit entries.

RFID tags (10-digit IDs)

Map each tag to an effect. Keys must be ten digits (leading zeros count). Kinds: reveal_letter (random hidden slot on a 5-letter lock), reveal_digit (random hidden digit on a 3- or 4-digit lock), hint / punishment with optional message. Easier difficulties cluster reveals on fewer locks; harder spreads them.

GM — live session

Shows active combinations for the running game (keep off the player monitor).
